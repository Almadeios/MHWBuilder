Data Templates

This folder contains editable JSON templates you can paste into the app's data files.

- `compact-skill.json` — compact skill entry example for `src/data/compact/skills.json`
- `detailed-skill.json` — detailed skill metadata example for `src/data/detailed/skills.json`

- `compact-decoration.json` — compact deco example for `src/data/compact/decoration.json`
- `detailed-decoration.json` — detailed deco example for `src/data/detailed/decoration.json`

- `compact-armor.json` — compact armor example (use in the appropriate slot file)
- `detailed-armor.json` — detailed armor example (use in the appropriate slot file)

- `compact-talisman.json` — compact talisman example for `src/data/compact/talisman.json`
- `detailed-talisman.json` — detailed talisman example for `src/data/detailed/talisman.json`

- `id-maps.json` — examples for adding IDs to `src/data/ids/*`
- `talisman-rules.json` — template for `src/data/talisman-generator/rules.json`

Copy the snippet you need and paste it near the bottom of the matching JSON object in the repository.
